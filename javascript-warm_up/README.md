# JavaScript Warm Up

This directory contains introductory JavaScript exercises and challenges designed to help you get comfortable with JavaScript fundamentals.

## Contents

- Basic syntax and variables
- Functions and scope
- Control flow statements
- DOM manipulation basics
- Event handling

## Getting Started

1. Open the exercise files in your editor
2. Follow the comments and instructions in each file
3. Test your code in the browser or Node.js environment
